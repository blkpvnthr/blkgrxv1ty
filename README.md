# Project BLKGR4V1TY OS

A personal operating system for career strategy, job tracking, portfolio development, capital planning, and personal brand growth.

## Stack

- React
- Vite
- Firebase Hosting
- Firebase Analytics
- Firestore/Auth-ready architecture
- LocalStorage persistence for V1

## Setup

```bash
npm install
cp .env.example .env
```

Paste your Firebase values into `.env`.

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Firebase

```bash
npm install -g firebase-tools
firebase login
firebase use blkgrxv1ty
npm run deploy
```

## V1 Features

- BLKGR4V1TY dashboard
- Interactive 30-day roadmap
- Job tracker
- Portfolio tracker
- Capital and personal brand placeholders
- Local browser persistence
